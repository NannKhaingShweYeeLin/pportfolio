    In this project we are creating Users can be Employees, Managers or Admins, Notes can only be deleted by Managers or Admins, Anyone can create a note ( when customer checks-in), Employees can only view and edit their assigned notes, Managers and Admins can view, Edit, and delete all notes, only Managers and Admins can access User Settings. This project is created with MongoDB for saving data and fetching data, Express jS, Note Js and React Js. In this project I'm in frontend and backend part.
